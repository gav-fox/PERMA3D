# PERMA3D

Computational agroecology. Open source, AGPLv3.

A desktop tool that ingests real terrain (DEM) and climate (EPW) data for a single parcel and uses parametric design plus evolutionary algorithms to generate honest, site-specific regenerative farm designs.

## Status

Early development. Engine 1 (Site Reader) in progress.

## Test Site

Velez-Rubio, Almeria, Spain. 20 hectares of degraded inland highland at ~840m. BSk cold semi-arid climate.

## Honest limits

- Simulations are plausible, not verified.
- This is a design generator, not a replacement for observing the land.
