# PERMA3D

Computational agroecology. Open source, AGPLv3.

A desktop tool that ingests real terrain and climate data for any parcel of land and uses parametric design and evolutionary algorithms to generate honest, site-specific regenerative farm designs in 3D.

## What it does

Takes topography (DEM) and climate (EPW) data as inputs. Computes terrain layers -- slope, aspect, water flow, frost risk, sun hours, wind exposure. Uses that to score and evolve permaculture designs against real site conditions.

## Status

Early development. Engine 1 (Site Reader) in progress.
